PictureBoxExercise 🖼️

An interactive C# Windows Forms application that demonstrates dynamic UI updates using RadioButtons, Labels, and Embedded Resources.
💡 Project Overview

This project is an educational exercise focused on building a responsive desktop interface. It allows users to switch between different categories (Boy, Girl, Book, Pen). When a category is selected, both the image and the title label update instantly.
🛠 Technical Details

    Framework: .NET Framework 4.7.2.

    Language: C#.

    Key Controls: RadioButton, PictureBox, Label.

🚀 Learning Objectives

    Event Handling: Using CheckedChanged events to trigger UI changes.

    Resource Management: Loading images (PNG/JPG) directly from the project's internal resources (Properties.Resources).

    UI Synchronization: Keeping the text (Label) and graphics (PictureBox) in sync based on user input.
